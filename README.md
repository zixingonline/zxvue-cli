# zixing-cli
A simple webpack-vue scaffold / 一个简单干净的webpack-vue自定义脚手架工具
## Requirements
* Node尽量使用最新版本
* Webpack >= 4.0

## On the command line
### Install
```
npm install zxvue-cli -g
```
### Use
```
zxvue-cli init <Project Name>
cd <Project Name>
npm install
```
